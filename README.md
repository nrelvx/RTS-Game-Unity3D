# RTS Game (Unity 6)

A real-time strategy (RTS) game prototype developed in **Unity 6** using **C#**.

## About

This project is a personal learning project focused on developing the core mechanics of an RTS game. The goal is to implement a modern RTS camera system, unit interaction, building placement, resource management, and other gameplay features.

## Features

* 🎮 RTS camera controls
* ⌨️ WASD camera movement
* 🔄 Camera rotation (Q / E)
* ⚡ Camera speed boost (Left Shift)
* 🌍 Low Poly environment
* 🛠️ Built with Unity 6 and C#

## Technologies

* Unity 6
* C#
* Universal Render Pipeline (URP)
